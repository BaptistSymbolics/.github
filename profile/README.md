# Baptist Symbolics

Open-source editions of historic Baptist confessional and catechetical documents, maintained in structured data formats (TOML) with version-controlled history and PDF generation.

All texts are freely available under the [CC0 1.0 Universal](http://creativecommons.org/publicdomain/zero/1.0/) license.

## About

These projects aim to:
- Preserve important Baptist theological documents in accessible, structured formats
- Provide conservative modernizations that update archaic language while preserving theological content
- Enable community collaboration through version control and open licensing

Contributions, corrections, and feedback are welcome. Please open an issue in the relevant repository.
